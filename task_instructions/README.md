# Instructions Used To Create

1. From https://docs.amd.com/v/u/en-US/ug1676-sdkdoc-platformstudio-documentation download each ZIP file to the top directory of the repo. 

> [!NOTE]
> Please download the Platform Studio ZIP files in ascending order, then the SDK ZIP file in ascending order. 

2. Add the date and time the ZIP file was downloaded.
3. Extract the zip file so that the directory containing the files resembles `old_sdk_docs\platform_studio_10\platform_studio\`.
4. Add a link to the landing page of the new repo under the README.md's index.
5. Check in the ZIP file, the expanded directory, and the README.md. Use a subject like `Add platform_studio_10`. Do one check-in per ZIP file.
