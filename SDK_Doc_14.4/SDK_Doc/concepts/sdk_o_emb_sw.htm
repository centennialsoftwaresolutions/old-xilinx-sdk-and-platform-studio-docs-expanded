Xilinx embedded software

Setting up your embedded software development for the Xilinx embedded processors in SDK involves the following:

  1. Creating an SDK Board Support Package (BSP) project and configuring it appropriately for your application
  2. Creating an SDK Software Application project and adding your source code and build flags to it
  3. Creating an SDK BSP for External Tools project and exporting the generated BSP into the third party environment (if your application development is being done in a third-party environment)
