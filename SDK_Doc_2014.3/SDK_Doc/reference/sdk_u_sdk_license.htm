Obtaining SDK Licenses

Beginning with the 2014.1 release, a license is required for running SDK.

When SDK is launched, it checks for a valid “SDK Feature” license. Two types of licenses are available:

  • Evaluation/30 Day Trial License: This license allows you to run SDK in Evaluation mode. In this mode, you can access all of the SDK features except the SDK Performance feature.
  • Full License: This license allows you to access all SDK features.

If you do not have an SDK license, SDK displays the following error message and then exits. You must re-launch SDK again after obtaining a license.

Using the Vivado License Manager

When you install SDK, the Vivado License Manager is also installed on your system and runs at the end of the installation.

You can also run the Vivado License Manager by selecting Manage Xilinx Licenses from the SDK or Vivado folder in the Start menu.

Acquiring a License

The following steps lead you through acquiring a license. If you have already acquired a license and need to load it, refer to Loading an Existing License

  1. Click Obtain License on the left side of the window.
  1. Click one of the following:
    • Click the Start 30 Day Evaluation option to create an evaluation license.
    • Click Get my Purchased License(s) to create a full license or to get an already purchased license.
  2. Click Connect Now.

The Xilinx Licensing website opens.

  1. Sign into your account, or register for a new account if necessary.
  2. Follow the website prompts to create your license.
Loading an Existing License

The following steps lead you through loading an existing license.

  1. Click Load License on the left side of the window.
  1. If you have a certificate-based license, click Copy License, and then navigate to your .lic file to load it.
  2. If you have an activation-based license, click Activate License, and then navigate to your XML file to activate your machine for Xilinx tools and IP.
