Zynq DRAM Diagnostics Test

The Zynq DRAM diagnostics test is a stand-alone program running on a single Zynq Cortex-A9 processor, executing out of OCM. The program uses the UART for interactive operation.

The full Zynq DRAM Diagnostics Test instructions can be accessed in your SDK installation by clicking here or navigating to <SDK installation directory>\data\embeddedsw\lib\sw_apps\zynq_dram_test\src .
