XSDB Commands: Registers

The Xilinx System Debugger CLI (XSDB) Registers commands can be used to Read and Write to the target registers.

The following commands are available:

  • rrd
  • rwr

These commands can access General Purpose registers, System registers like ARM Coprocessor registers. IOU registers in Zynq can be accessed by running these commands on 'APU target'.

'rrd' can also read register definitions, instead of their values.

rrd

Description

Read register definitions or values for the active target.

Usage

rrd [options] [reg]

Read registers or register groups for active target.

Options

-defs

Read register definitions instead of values

Examples

rrd

Read top level registers or groups

rrd r0

Read register r0

rrd usr r8

Read register r8 in group usr

Rrd -defs

Read definitions for top level registers or groups
rwr

Description

Write to a register on active target

Usage

rwr <reg> <value>

Write the <value> to active target register specified by <reg> .

<reg> can specify a top level register and a register within a group.

Examples

rwr r8 0x0

Write 0x0 to register r8 .

rwr usr r8 0x0

Write 0x0 to register r8 in group usr
