Standalone Application Debug Using Xilinx System Debugger

This topic describes how to use the Xilinx®-customized System Debugger to debug bare-metal applications.

  1. Create a sample Hello World project.
  1. Select the application and click Run > Debug As > Launch on Hardware System Debugger.
