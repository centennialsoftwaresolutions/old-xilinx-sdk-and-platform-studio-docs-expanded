System Performance Modeling Using a User-Defined Flow

Performance Analysis can be done on any user-defined applications.

To start performance analysis for the application, right-click the application and select Debug As > Start Performance Analysis .

If no ATG is configured in the Hardware, the ATG Configuration window will be empty. Make sure to remove the Breakpoints by selecting Window > Show View > Breakpoints .
