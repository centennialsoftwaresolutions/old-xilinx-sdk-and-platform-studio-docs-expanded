What’s New in SDK

The following updates were made in SDK 2014.3:

General
  • Eclipse version was migrated to Kepler (version 4.3.2)
  • Enhanced the XSDB command-line debugger
  • Device programming is now through UpdateMEM for 7 series devices
  • New SDK batchmode implemented, which includes complete hardware software interface (HSI) command set.
  • You can now automatically create driver example projects from MSS files.
Performance Analysis
  • New Performance perspective, which is common for both ZPE and User designs
  • ZPE is now integrated in SDK for the Zynq-7000 ® ZC702 platform
  • Enhanced PS and PL performance analysis through tabular and graphical views for any user design.
    • Support for performance capture over 4 HP ports and one ACP port
    • CPU Utilization, IPC, Cache hit/miss
    • Zoom in feature allows you to zoom into an area
    • Configuration of ATG in run time
MicroBlaze Debug
  • Non-intrusive debug access to system peripherals and memory
  • Beta version of MB performance analysis
    • CPU Utilization
    • Instructions per Cycle data
    • Cache hit/miss rates

 

For changes in earlier releases, refer to What’s New in SDK 2014.2 and Earlier Releases.
