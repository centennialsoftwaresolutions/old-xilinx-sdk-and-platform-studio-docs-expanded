What’s New in SDK

Updates

The following updates were made in SDK 2014.2:

  • Create Bootgen now includes edit mode.
  • Webtalk is now integrated in the Vivado® Design Suite + SDK package.
  • Support for a new hardware handoff file (*.hdf), as well as existing format of XML.
  • New feature to support Linux application creation by taking the sysroot of the Linux file system, and an option to provide a selected tool chain.
  • Several issues are fixed in this release.

For changes in earlier releases, refer to What’s New in SDK 2014.1 and Earlier Releases.
