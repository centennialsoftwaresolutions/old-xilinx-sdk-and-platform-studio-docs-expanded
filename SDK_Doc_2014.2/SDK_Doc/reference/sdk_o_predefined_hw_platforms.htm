Pre-Defined Hardware Platforms

At present, there are three pre-defined hardware platforms for ZC702, ZC706, ZED, and MicroZed hardware. With these, you can start your application on-the-fly instead of using hardware design creation tools. These are specific to the PS-only part of Zynq®-7000 AP SoC devices.
