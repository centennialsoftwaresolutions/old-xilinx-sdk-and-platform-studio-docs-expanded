System Debugger Tcl Mode (XSDB)

2014.1 is the Alpha release of the TCF command line interface. The TCF connection is a connection to the hardware server in the host/remote machine.

The sequence of commands for the TCF connection is as follows:

  1. Help (Provides the entire help)
  2. connect (establishes connection to hw_server)
  3. targets (displays the debug devices on the target)
  4. target < target Id > (1/2/3 or 4 selects the device for debug)
  5. state (Gives processor state)
  6. mrd/mwr (same as earlier XMD command format)
  7. rrd (processor registers)
  8. stop (stops the processor)
  9. con (resume from the current PC value)
