Creating or Editing a Run/Debug/Profile Configuration

Running an Application

You can run an application by right-clicking the application name in the C/C++ Projects View and clicking Run As > Launch on Hardware (GDB). This creates a default run configuration and runs the application on the target. You can also configure a run environment with which to run your application.

Debugging an Application

To debug an application using the default run and debug settings:

  1. Right-click on the application
  2. Select Debug As > Launch on Hardware (GDB) or Launch on Hardware (System Debugger), based on the requirement of your design.
