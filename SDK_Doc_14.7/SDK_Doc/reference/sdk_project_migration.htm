Migrating SDK 12.x Projects to SDK 14.x and Higher

Migrating from SDK 12.x to SDK 14.x and higher versions should be seamless in most cases.

There is one manual step involved if your projects use Cygwin-style paths. In such cases, these paths must be converted to use Windows-style or relative paths. SDK 13.x does not include Cygwin, and the GNU toolchain used is built on MingW.
