IPIF Features
The IPIF features are described in the following table:
IPIF FeatureDescription
Software reset For PLB v4.6 peripheral only. The peripheral has a special write-only address. When a specific word is written to this address, a reset signal is generated and the peripheral will reset itself. This allows individual peripheral to be reset from the software application.
Read/Write FIFOFor PLB v4.6 peripheral only. The peripheral has a built-in First In First Out (FIFO) service by directly instantiating the FIFO components from EDK built-in IP library.
Interrupt controlFor PLB v4.6 peripheral only. The peripheral has an interrupt collection mechanism that manages the interrupts generated by the user logic and other services included in the peripheral and generates a single interrupt output line from the peripheral.
User logic software registerThe user logic component of the peripheral has registers that are addressable through software.
User logic memory spaceGenerates enable signals for each address range. This feature is useful for peripherals that need to support multiple address ranges, such as multiple memory banks. The distinction between this and other cases is that the enable signals are generated for each address range of the address space supported by the peripheral, rather than for each addressable register in the user-logic module.
Include data phase timerThis indicates if the data phase timer is used or not. The value of 0 excludes the timer. The value of 1 includes the timer. If C_INCLUDE_DPHASE_TIMER = 1 and after 128 SPLB_Clk cycles, as measured from the assertion of Sl_AddrAck, the IP does not respond with either an IP2Bus_RdAck or IP2Bus_WrAck. The plbv46_slave_single de-asserts the IP cycle request signals, Bus2IP_CS and Bus2IP_RdCE or Bus2IP_WrCE, and asserts Sl_rdDAck with Sl_rdDBus=zero for a read cycle or Sl_wrDAck for a write cycle. This gracefully terminates the cycle.
Note The requesting master has no knowledge that the data phase of the PLB request was terminated in this manner.
User logic masterThis includes the IPIC master interface signals for user-logic master operations. It also includes the example Hardware Description Language (HDL) of a simple master operation model.
