Adding Libraries and Library Paths

You can add libraries and library paths for Managed Make projects. If you have a custom library to link against, you should specify the library path and the library name to the linker.

To set properties for your project:

  1. Right-click your managed make project and select C/C++ Build Settings.
     Alternately, you could select Properties and navigate to C/C++ Build > Settings.
  2. In the Tool Settings tab, expand the target linker section and select the libraries to which you want to add the custom library path and library name.
