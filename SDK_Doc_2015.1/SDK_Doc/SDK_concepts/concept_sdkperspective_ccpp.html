C/C++ Perspective Views

The C/C++ Perspective provides a collection of views to develop C and C++ projects. By default, it consists of an editor area and the following views:

  • SDK Projects (tree view of hardware and software projects)
  • C/C++ Projects (the file navigator for C/C++ resources)
  • Navigator (the file navigator for all Eclipse resources)
  • Console
  • Properties
  • Tasks
  • Make Targets
  • Outline
  • Search
