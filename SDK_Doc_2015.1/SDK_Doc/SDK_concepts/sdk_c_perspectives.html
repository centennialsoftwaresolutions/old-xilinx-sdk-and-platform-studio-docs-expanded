Perspectives Available to C/C++ Developers in SDK

A perspective is a layout of views (development tools) in the Workbench window. Each type of perspective is a combination of views, menus, and toolbars that enable you to perform a particular task. For example, the C/C++ Perspective has views that are organized to help you develop C/C++ programs, and the Debug Perspective has views that enable you to debug those programs. For more information, refer to Views in the C/C++ Perspective.
