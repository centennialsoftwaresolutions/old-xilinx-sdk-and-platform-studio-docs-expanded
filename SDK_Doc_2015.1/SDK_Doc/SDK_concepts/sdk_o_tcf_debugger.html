Debugging with System Debugger

Apart from the Xilinx-customized System Debugger, the XMD/GDB debug flow that is supported in earlier versions of the tool can still be used, but this flow is deprecated.

System Debugger supports all Xilinx processors and JTAG cables. The links below provide pointers to topics on installation, setting up the environment, and debugging applications.
