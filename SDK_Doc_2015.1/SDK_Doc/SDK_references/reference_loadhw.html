loadhw

Description

Load a Vivado HW design.

Usage

 loadhw [Options]

Load a Vivado HW design, and set the memory map for the current target. If the current target is a parent for a group of processors, memory map is set for all the child processors. If the current target is a processo memory map is set for all the child processors of it's parent.

This command returns the HW design object.

Options

 -hw

Specify the HW design file.
