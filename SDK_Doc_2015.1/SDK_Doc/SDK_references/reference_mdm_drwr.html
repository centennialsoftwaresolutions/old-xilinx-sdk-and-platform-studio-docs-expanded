mdm_drwr

Description

Write to MDM Debug register.

Usage

 mdm_drwr [OPTIONS] <cmd> <data> <bitlen>

Write to MDM Debug Register.

  • <cmd> is 8-bit MDM command to access a Debug Register.
  • <data> is the register value.
  • <bitlen> is the register width.

Options

 -user <bscan number>

Specify user bscan port number. Default is 2.
