mrd

Description

Read memory address of the active target.

Usage

 mrd [-force] <address> [num]

Read <num> words from active target memory address specified by <address>

Examples

 mrd 0x0

Read a word at 0x0
mrd 0x0 10

Read 10 words at 0x0
mrd -force 0xe0000000

Over-write access protection and read a word at address 0xe0000000
