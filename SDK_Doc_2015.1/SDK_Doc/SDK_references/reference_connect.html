connect

Description

Connect to hw_server. Unlike with XMD, the connect command in XSDB doesn't connect to a debug target on the hardware. You must use the "targets" command to list the available targets and select a debug target before issuing commands to the debug target.

If no options are used with the connect command, XSDB launches hw_server at localhost:3121 and connects to it.

The connect command returns the channel ID of the connection to the hw_server.

Usage

 connect [Options]  

Options

 -host <host name/ip>

Name/IP address of the host machine
-port <port num>

TCP port number
-url <url>

URL description of hw_server
-list

List open connections
-set <channel-id>

Set active connection

Examples

connect -host localhost -port 3121

Connect to hw_server on host localhost and port 3121

connect -url tcp:localhost:3121

Identical to previous example
