unloadhw

Description

Unload a Vivado HW design.

Usage

 loadhw [Options]

Close the Vivado HW design which was opened during loadhw command, and clear the memory map for the current target. If the current target is a parent for a group of processors, memory map is cleared for all the child processors. If the current target is a proc memory map is cleared for all the child processors of it's parent.

This command doesn't clear the memory map explicitly set by users.
