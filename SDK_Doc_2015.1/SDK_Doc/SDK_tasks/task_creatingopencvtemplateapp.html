Creating an OpenCV Template Application

To create an OpenCV template application:

  1. Select New > Application Project. The New Project wizard opens and displays the Application Project page.




  2. In the Processor drop-down box, select the processor of any ARM core.
  3. In the OS Platform drop-down box, select Linux.
  4. Click Next. The Templates page opens.
  5. Select the template type OpenCV Example Application.




  6. Click Finish. SDK creates an OpenCV example application. You can edit and change this application as required.
  7. You can add additional header files. To do this:
    1. Right-click on the project name and select C/C++ Build Settings. The Properties dialog box opens, with the C/C++ Build > Settings pane selected.
