Creating a Run Configuration

You can create a customized run configuration and save it for reuse.

To create a run configuration:

  1. In the C/C++ Projects perspective, select a project.
  2. Select Run > Run Configurations. Note: Alternately, right-click on the application and select Run > Run Configurations.
  3. In the Configurations list, click one of the following:
  4. Xilinx C/C++ application (System Debugger) for standalone application
  5. Remote ARM Linux Application for Linux based application
  6. Click New. The name of the new project displays in the Configurations list.
