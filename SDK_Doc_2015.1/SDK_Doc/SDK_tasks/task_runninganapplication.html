Running an Application

The Run As option downloads the application to the target and immediately begins execution.

  1. Right-click the application.
  2. Select Run As.
  3. Select Launch on Hardware (System Debugger).

The last option that appears in the pop-up menu is Run/Debug/Profile Configurations, which allows you to override the default values and select specific values for this particular task. For more details, refer Run/Debug Configuration Settings.
