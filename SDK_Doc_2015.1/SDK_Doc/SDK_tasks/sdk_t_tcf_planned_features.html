SDK System Debugger

This topic provides the details that are needed to start using the new, customized System Debugger integrated into Xilinx SDK.

The System Debugger from open source is customized and is integrated with Xilinx® SDK.

Apart from the Xilinx-customized System Debugger, the XMD/GDB debug flow that is supported in earlier versions of the tool can still be used. Users will see an improved performance in this release due to the updates in XMD and underneath cable infrastructure to support coalescing of multiple commands and the usage of System Debugger agent for communication.

The System Debugger support is integrated only for the ARM-over-Digilent cable. You can use the SDK-XMD flow for the Xilinx USB cable, which will continue to work in the current release. The scope of this document focuses on the new, customized System Debugger over Digilent cable for ARM.
