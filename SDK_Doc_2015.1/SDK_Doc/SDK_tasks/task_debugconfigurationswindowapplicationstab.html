Debug Configurations Window, Applications Tab

  1. Select the Applications tab to display all processors available to the selected hardware platform.
  2. Select the Download Application checkbox if you want to download the application to the selected processor. Note: If a single project exists for the processor, the Project Name and Application Name fields populate automatically when you select the Download Application checkbox. If more than one project exists for the processor, you must make the Project Name selection manually.
  3. Select the Stop at program entry checkbox if you want to stop the processor before application main().




  4. Click the Debug button to launch multi-processor debugging.
