breakpoints - Target Breakpoints/Watchpoints

The Xilinx System Debugger CLI (XSDB) Breakpoints commands can be used to add, remove, enable, disable or list Breakpoints/Watchpoints. XSDB supports Breakpoints/Watchpoints at address and source line.

The following commands are available:

  • reference_bpadd.html#sdk_u_xsd_xsdb_breakpoints
  • reference_bpremove.html#sdk_u_xsd_xsdb_breakpoints
  • reference_bpenable.html#sdk_u_xsd_xsdb_breakpoints
  • reference_bpdisable.html#sdk_u_xsd_xsdb_breakpoints
  • reference_bplist.html#sdk_u_xsd_xsdb_breakpoints
