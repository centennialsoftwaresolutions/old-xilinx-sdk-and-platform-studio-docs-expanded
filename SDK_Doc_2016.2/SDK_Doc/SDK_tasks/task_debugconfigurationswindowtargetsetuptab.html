Target Setup Tab

  1. In the Target Setup tab, select the appropriate setup.
  2. Select the Standalone Application Debug from the Debug Type dropdown list.
  3. Select the target you want to connect. With this selection, no resets or initializations are performed on the target before launching the debugger.
  4. To automatically populate bitstream and initialization files, from the Hardware platform drop-down list, select the appropriate hardware platform. Use the Browse buttons if you wish to select different bitstream and initialization files.


  5. If you want to reset the entire system, enable the Reset entire system checkbox in the Debug Configurations window.
  6. If you want to program the bitstream after system reset, enable the Program FPGA checkbox.
  7. Enable the Run ps7_init checkbox to run the PS initialization file. Note: The Summary window displays a summary of System Debugger operations.
