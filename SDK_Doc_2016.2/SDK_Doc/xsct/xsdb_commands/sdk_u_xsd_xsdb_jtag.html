JTAG Access

These commands can be used to perform raw JTAG shifts, get/set device properties, lock/unlock JTAG cable, etc.

The following is a list of jtag commands:

  • jtag targets
  • jtag sequence
  • jtag device_properties
  • jtag lock
  • jtag unlock
  • jtag claim
  • jtag disclaim
  • jtag frequency
  • jtag servers
