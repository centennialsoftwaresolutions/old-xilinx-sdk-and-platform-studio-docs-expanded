XSDB Console

Console view for the XSDB process launched by SDK. SDK uses XSDB for programming FPGAs, initializing Jtag devices, downloading programs, etc. The commands entered by users in this console are sent to the XSDB process and the output from the XSDB process is displayed in this console.

For more information, refer Xilinx System Debugger Command-line Interface (XSDB).
