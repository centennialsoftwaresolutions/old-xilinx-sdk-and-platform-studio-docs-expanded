rwr

Register Write

Syntax

rwr <reg> <value>

Write the <value> to active target register specified by <reg>

Returns

Nothing, if successful. Error string, if an invalid register is specified or the register can't be written.

Example(s)

rwr r8 0x0

Write 0x0 to register r8 

rwr usr r8 0x0

Write 0x0 to register r8 in group usr
