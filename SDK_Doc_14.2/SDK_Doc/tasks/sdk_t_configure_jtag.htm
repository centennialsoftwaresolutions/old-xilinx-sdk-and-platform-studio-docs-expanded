Configuring the JTAG settings

Note: In most cases, the default JTAG settings will work for your project and you can skip this task.

You might need to configure the JTAG settings for your project if you have any of the following:

  • Multiple JTAG cables connected to your machine
  • Using 3rd party cable that supports Xilinx cable plug-in mechanism
  • Multiple boards connected to your machine using different cables
  • A custom development board with non-Xilinx devices

To configure the JTAG settings:

  1. Select Tools > Configure JTAG Settings to open the Configure JTAG Settings dialog box.
  2. In the JTAG Cable area of the dialog box, specify the JTAG cable type, port (on the host machine), and frequency.
  3. If you have a 3rd party cable, select 3rd Party Cable, Xilinx Plug-in type.
  4. Specify the cable name and arguments (optional) to the plug-in in the Other Options field. You should specify the complete cable option -cable type xilinx_plugin modulename <cable name> {modulearg <cable args>} text in this field.
  5. The default JTAG device chain option setting is Automatically Discover Devices on JTAG Chain. With this option, the tools will automatically discover the JTAG devices for you.
    Alternately, you can select Manual Configuration of JTAG Chain to specify a non-Xilinx device on the chain. When you select this option, the JTAG Device Chain list becomes available. You can use the device buttons to manage your devices:
    • Add Device button: Adds a device to the list.
    • Delete Device button: Deletes the selected device from the list.
    • Move Up Device button: Moves the selected device up in the list.
    • Move Down Device button: Moves the selected device down in the list.
  6. Select the FPGA device in the JTAG chain using the FPGA? field. If you have multiple FPGA devices in the chain, select the one where the hardware design is loaded.
  7. Click OK.

Note: If you are debugging a PowerPC 405 or 440 processor that has JTAG pins are connected to the FPGA I/O pins, select Manual Configuration of JTAG Chain, specify the Device Name and ID Code of the FPGA device and specify the IR Length as ‘4’.
