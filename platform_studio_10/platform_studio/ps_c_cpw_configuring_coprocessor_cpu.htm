Configuring a Coprocessor for a CPU
Configuring a coprocessor for a CPU involves the following tasks.
Launching the Coprocessor Tool
Launch the Coprocessor tool by selecting Hardware > Configure Coprocessor.
The table on the right lists all coprocessors that are available in the repository. The left table lists the instance names of the coprocessors that are attached to the CPU. On exiting this tool, the coprocessor is instantiated in the system and connected to the central processing unit (CPU) through one or two Fast Simplex Link (FSL) buses, depending on how many FSL interfaces the coprocessor has.
Attaching a Coprocessor to the CPU
Attach a Coprocessor to the CPU, as follows:
  1.  Select the coprocessor you want to attach in the right hand side table.
  2.  Click Add.
Note You can also drag the coprocessor and drop it in the table on the left. After the coprocessor is added, you can double-click it to connect or disconnect each individual bus interface. On exiting from this tool, the system under construction is updated with the new coprocessor instantiated in the system and wired to the selected CPU instance.
Adding a Coprocessor to the Peripheral Repository Directory
Add a Coprocessor to the Peripheral Repository Directory, as follows:
  1.  Select Project > Project Options.
  2.  With the Device and Repository tab selected, specify the location of the peripheral in the Peripheral Repository Directory field provided.
Note Alternatively, copy the whole coprocessor to a peripheral repository directory that is visible to the current project.
Removing a Coprocessor from a CPU Instance
To remove a Coprocessor from a CPU Instance, select the coprocessor you want to remove and click Remove.
Alternatively, you can drag the coprocessor and drop it in the table on the right. As a result, the coprocessor instance is removed from the system along with the FSL buses that connect the coprocessor and the CPU.
Permanently Deleting a Coprocessor
To permanently delete a coprocessor, delete the directories under the drivers and pcores directory in the XPS project directory or the IP repository directory.
Creating a New Coprocessor
Create a New Coprocessor. You create a new coprocessor in the Create and Import Peripheral wizard. To launch the wizard:
  1.  Run the command creatip or select Hardware > Create or Import Peripheral.
  2.  In the Bus Interface page, specify the type of bus interface you want your peripheral to have.
  3.  Select the Fast Simplex Link option on that page.
