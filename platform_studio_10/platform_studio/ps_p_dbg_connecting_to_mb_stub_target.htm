Connecting to a MicroBlaze Stub Target in the Debug Options
To connect to a MicroBlaze™ Stub target:
  1.  In the Connection Type group box, select Stub.
  2.  In the Advanced Options tab, select the JTAG option to communicate with XMDSTUB over the JTAG interface.
  3.  In the JTAG Properties tab, select JTAG Cable Type and Speed.
  4.  In the JTAG Properties tab, use the Auto-Discover JTAG Chain Definition option to let XMD automatically discover the JTAG chain on the board and the FPGA device to debug.
  5.  If XMD cannot automatically discover the JTAG chain or the correct FPGA device, use the JTAG Options group to add or delete devices to the JTAG chain in the same order in which it is present on the board. To do this:
    1.  Use the Add button to add a device. Enter Device Name, Device ID code, and Instruction Register Length.
    2.  Use the Delete button to delete device. Select the device row and click Delete Device.
    3.  Select the device and use the up and down buttons in the JTAG options group to reorder devices in the JTAG chain.
  6.  Click Save.
  7.  In the Advanced Options tab, do the following:
    1.  select the Serial button to communicate with XMDSTUB over Serial interface.
    2.  In the Advanced Options tab, select the serial port used to communicate.
    3.  In the Advanced Options tab, select the Baud Rate value.
  8.  Click Save.
