Run or Debug Configurations dialog box

The Run or Debug Configurations dialog box allows you to create a run or debug configuration that can be used to run or debug your design in SDK.

You can also profile your design using this dialog box.

The Run or Debug dialog box contains a series of tabs, which are described in the following topics:

  • Selecting an application to run/debug/profile
  • Specifying initialization tasks
  • Specifying STDIO connections
  • Specifying XMD target configuration
  • Specifying Profiler configuration
