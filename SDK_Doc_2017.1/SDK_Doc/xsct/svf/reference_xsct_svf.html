SVF Operations

Serial Vector Format (SVF) is an industry standard file format that is used to describe JTAG chain operations in a compact, portable fashion. SVF files are portable because complicated vendor-specific programming algorithms can be conveyed by generic SVF instructions, requiring no special knowledge of the target device. XSCT provides commands that can directly generate SVF files for memory accesses, downloading elfs to memory, stop/start the core, reset the core etc.

Note: This feature is currently available only for ARM® Cortex-A53 cores.

The following is a list of svf commands:

  • svf config
  • svf generate
  • svf mwr
  • svf dow
  • svf stop
  • svf con
  • svf delay
