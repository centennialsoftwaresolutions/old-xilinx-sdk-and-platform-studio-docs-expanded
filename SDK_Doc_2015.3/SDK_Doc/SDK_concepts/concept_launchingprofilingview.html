Viewing the Profiling Results

Once the program completes execution (reaches exit), or when you click the Stop button to stop the program, SDK downloads the profile data and stores it in a file named gmon.out.

To view the profile results, double click on the gmon.out file in the Project Explorer view (or drag the file into the editing area).

TCF profiler view is updated periodically with the profile samples. Clicking on a sample in the profiler view will show the corresponding function in the source view. Various columns in the profiler view are described in the table below.

Column

Description

Address

Address in the application at which the samples were collected.

% Exclusive

Amount of time spent in this function. This is exclusive of the samples collected in child functions.

% Inclusive

Amount of time spent in this function, including the time spent in child functions.

Function

The function in which the sample was collected.

File

Name of the file for which the sample was collected.

Line

Line number for which the sample was collected.
