Cross-Triggering in Zynq

In Zynq®, ECT is configured with four broadcast channels, four CTIs, and a CTM (for example, see Cross-Triggering). One CTI is connected to ETB/TPIU, one to FTM and one to each Cortex®-A9 core. The table below shows the trigger input and trigger output connections of each CTI.

Note: The connections specified in the table below are hard-wired connections.
