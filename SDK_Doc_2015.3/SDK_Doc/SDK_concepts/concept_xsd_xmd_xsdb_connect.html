connect

In both XMD and XSDB, connect is the primary command to establish connections to debug targets. However, there is a difference in how connect works in XMD and XSDB.

XMD provides the connect command to connect to a debug target, such as a processor target.

In XSDB, the connect command opens a connection to the hw_server. Use the targets command to list all the available targets and select a debug target before issuing any debug commands to that target.
