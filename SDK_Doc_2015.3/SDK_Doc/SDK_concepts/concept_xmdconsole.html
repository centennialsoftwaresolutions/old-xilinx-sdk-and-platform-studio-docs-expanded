XMD Console

Console view for the XMD process launched by SDK for use with GDB debug session. The commands entered in this console are sent to the XMD process and the output from the XMD process is displayed in this console.

Note: Disconnecting from targets/Jtag cables, while a GDB debug session is active can cause SDK to lose sync with the target state.
