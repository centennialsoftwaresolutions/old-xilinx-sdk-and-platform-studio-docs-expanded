SDK Batchmode Commands

Use the Xilinx Software command-line tool to perform various SDK related tasks.

The following commands are available:

  • sdk set_workspace
  • sdk get_workspace
  • sdk set_user_repo_path
  • sdk get_user_repo_path
  • sdk get_app_templates
  • sdk create_hw_project
  • sdk create_bsp_project
  • sdk create_app_project
  • sdk build_project
  • sdk clean_project
  • sdk import_projects
  • sdk import_sources
