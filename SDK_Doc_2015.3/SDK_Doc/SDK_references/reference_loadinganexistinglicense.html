Loading an Existing License

The following steps lead you through loading an existing license.

  1. Click Load License on the left side of the window.



    • If you have a certificate-based license, click Copy License, and then navigate to your .lic file to load it.
    • If you have an activation-based license, click Activate License, and then navigate to your XML file to activate your machine for Xilinx tools and IP.
