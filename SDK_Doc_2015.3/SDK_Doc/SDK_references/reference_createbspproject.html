sdk create_bsp_project

Description

You can use this command to create a Board Support Package (BSP) project.

Make sure that you set the workspace before using this command. Specify the hardware platform project available in the workspace.

Usage

 sdk create_bsp_project [options]

Create a BSP project.

Options

 -name <project-name>

Project name that should be created.
-proc <processor-name>

Processor instance that should be used for extracting the BSP project.
-hwproject <hw project name>

HW project for which the application or bsp project should be created.

This option is valid only for creating application/bsp projects.
-os <OS name>

OS type for the application project.

Default type is standalone.
-mss <MSS File Path>

MSS file path for creating a BSP file.

This option can be used for creating a BSP from user mss file.

Note: When .mss file is specified, the -processor and the -os options are ignored. The processor/os details are extracted from the .mss file.
