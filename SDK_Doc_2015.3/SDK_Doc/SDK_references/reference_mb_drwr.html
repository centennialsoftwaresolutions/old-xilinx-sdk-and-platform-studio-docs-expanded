mb_drwr

Description

Write to MicroBlaze debug register

Usage

 mb_drwr [OPTIONS] <cmd> <data> <bitlen>

Write to MicroBlaze debug register available on MDM.

  • <cmd> is 8-bit MDM command to access a Debug Register.
  • <data> is the register value.
  • <bitlen> is the register width.

Options

 -user <bscan number>

Specify user bscan port number. Default is 2.

Example

 mb_drwr 1 0x282 10

Write to MB Control register.
