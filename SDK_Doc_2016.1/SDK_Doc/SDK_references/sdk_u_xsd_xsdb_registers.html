registers - Target Registers

The Xilinx System Debugger CLI (XSDB) Registers commands can be used to Read and Write to the target registers.

The following commands are available:

  • rrd
  • rwr

These commands can access General Purpose registers, System registers like ARM Coprocessor registers. IOU registers in Zynq-7000 AP SoC devices can be accessed by running these commands on 'APU target'.

'rrd' can also read register definitions, instead of their values.

  • rrd
  • rwr
