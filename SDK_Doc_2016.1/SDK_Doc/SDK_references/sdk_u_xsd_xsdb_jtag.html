jtag - JTAG Access

The Xilinx System Debugger CLI (XSDB) JTAG commands can be used to perform raw JTAG shifts, get/set device properties, lock/unlock JTAG cable, etc.

The following commands are available:

  • jtag targets
  • jtag sequence
  • jtag device_properties
  • jtag lock
  • jtag unlock
  • jtag claim
  • jtag disclaim
  • jtag frequency
  • jtag servers
