running - Program Execution

The Xilinx System Debugger CLI (XSDB) Running commands can be used for program execution, target state, and disassembly.

The following commands are available:

  • backtrace
  • con
  • stop
  • state
  • stp
  • stpi
  • nxt
  • nxti
  • stpout
  • dis
