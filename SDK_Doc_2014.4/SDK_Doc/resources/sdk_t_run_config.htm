Run Configuration Settings

Standalone Application
  1. In the Name field, type a descriptive name for the launch configuration.
  2. In the Main tab, set the following fields.
    Note: If you opened this dialog box by right-clicking on an application name and selecting Run > Run Configurations, these fields are automatically populated for that application.
    1. In the Project field, type the name of the project containing the application that you want to run, or browse to it.
    2. In the C/C++ Application field, type the name of the executable that you want to run. You can also search for the file or browse to it.
  1. For Device Initialization , specify initialization tasks as per requirement
    1. In the Reset Type drop-down list, select the reset type.
    2. If your program is in Flash and you do not want to download the ELF program file, select Do not download program to memory.
    3. If you want to download any data files, add the data files to the download list, along with address at which you want to download them.
  2. If required you can also use the console as STDIO terminal. In STDIO Connection Tab, select connect STDIO to Console, and set the Port and BAUD Rate.
  3. Click Apply to save the settings.
  4. Click Run.
Linux Application

First, do the following in the Main tab:

  1. In the Project field, type the name of the project containing the application that you want to run, or browse to it.
  2. In the C/C++ Application field, type the name of the executable that you want to run. You can also search for the file or browse to it.
  3. In connection field, select New > General > SSH Only and click Next.
    1. In the New connection window, enter the Target board IP in the Host name field
    2. Type the connection name and description in the respective fields.
    3. Click Finish to create the connection.

The Connection field now shows the Target IP.

  1. In the Remote Absolute File Path for C/C++ Application field, type the path in Linux Target that contains the location of the Application to be loaded and run.
  2. In the Commands to execute before application field, type the commands that need to be executed before running the application.
    For example, as shown in the figure below, the application FastCorner_sdk.elf uses OpenCV libraries Path, which is set in this field.
  3. Click Apply to save the settings.
  4. Click Run.
