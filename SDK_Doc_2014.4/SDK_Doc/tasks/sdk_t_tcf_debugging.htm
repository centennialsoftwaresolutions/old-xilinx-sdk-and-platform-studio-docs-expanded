Debugging with the System Debugger

The System Debugger, from open source, is customized and integrated with Xilinx® SDK.

Apart from the Xilinx-customized System Debugger, the XMD/GDB debug flow that is supported in earlier versions of the tool can still be used.

The System Debugger is intended for use only with the ARM over Digilent cable. Refer to the Recommended Debugger Usage topic to determine when to use which debugger.
