XSDB Commands: Running

The Xilinx System Debugger CLI (XSDB) Running commands can be used for program execution, target state, and disassembly.

The following commands are available:

  • con
  • stop
  • state
  • stp
  • stpi
  • nxt
  • nxti
  • stpout
  • dis
con

Description

Resume the active target.

Usage

con

Resume execution of the active target.
stop

Description

Stop the active target.

Usage

stop

Suspend execution of the active target.
state

Description

Display the current execution state.

Usage

state

Display the current execution state of the active target.
stp

Description

Step into a line of source code.

Usage

stp [count]

Resume execution of the active target until control reaches instruction that belongs to a different line of source code. If a function is called, stop at first line of the function code. Error is returned if line number information is not available.

If <count> is greater than 1, repeat < count > times.

Default value of count is 1.
stpi

Description

Execute a machine instruction.

Usage

stpi [count]

Execute a single machine instruction. If instruction is function call, stop at first instruction of the function code.

If <count> is greater than 1, repeat <count> times.

Default value of count is 1.
nxt

Description

Step over a line of source code.

Usage

nxt [count]

Resume execution of the active target until control reaches instruction that belongs to a different line of source code, but runs any functions called at full speed.

Error is returned if line number information not available.

If <count> is greater than 1, repeat <count> times.

Default value of count is 1.
nxti

Description

Step over a machine instruction.

Usage

nxti [count]

Step over a single machine instruction. If instruction is function call, execution continues until control returns from the function.

If <count> is greater than 1, repeat <count> times.

Default value of count is 1.
stpout

Description

Step out from current function.

Usage

stpout [count]

Resume execution of current target until control returns from current function.

If <count> is greater than 1, repeat <count> times.

Default value of count is 1.
dis

Description

Disassemble Instructions.

Usage

dis <address> [num]

Disassemble <num> instructions at address specified by <address> .

The keyword "pc" can be used to disassemble instructions at current PC.

Default value for num is 1.

Examples

dis

Disassemble an instruction at the current PC value.

dis pc 2

Disassemble two instructions at the current PC value

dis 0x0 2

Disassemble two instructions at address 0x0
