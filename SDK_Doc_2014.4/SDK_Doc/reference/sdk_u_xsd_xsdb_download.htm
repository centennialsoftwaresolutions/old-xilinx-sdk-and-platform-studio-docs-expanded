XSDB Commands: Download

The Xilinx System Debugger CLI (XSDB) Download commands can be used to download elf/binary files to the target or configure FPGA.

The following commands are available:

  • dow
  • fpga
dow

Description

Download elf/binary files to the active target

Usage

dow [-clear] <file>

Download ELF file <file> to active target.

The 'clear' option can be used to clear non-loadable sections like bss, during ELF download

dow -data <file> <addr>

Download binary file <file> to active target address specified by <addr>

fpga

Description

Configure FPGA with a bitstream.

FPGA device should be selected through the 'targets' command before running fpga command.

Usage

fpga [options] <bitstream-file>

Configure FPGA with given bitstream

Options

-file <bitstream-file>

Specify file containing bitstream

-partial

Configure FPGA without first clearing current configuration
