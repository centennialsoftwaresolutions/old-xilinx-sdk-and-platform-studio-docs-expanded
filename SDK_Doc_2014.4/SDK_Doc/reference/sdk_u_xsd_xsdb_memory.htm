XSDB Commands: Memory

The Xilinx System Debugger CLI (XSDB) Memory commands can be used to Read and Write to the target memory space.

The following commands are available:

  • mrd
  • mwr

Note: The 'force' option can be used to with these commands to over-write access protection.

mrd

Description

Read memory address of the active target.

Usage

mrd [-force] <address> [num]

Read <num> words from active target memory address specified by <address>

Examples

mrd 0x0

Read a word at 0x0

mrd 0x0 10

Read 10 words at 0x0

mrd -force 0xe0000000

Over-write access protection and read a word at address 0xe0000000

mwr

Description

Write to the memory address of the active target.

Usage

mwr [-force] <address> <values> [<num>]

Write <num> words from list of <values> to active target memory address specified by <address>

Examples

mwr 0x0 0x1234

Write 0x1234 to address 0x0

mwr 0x0 {0x12 0x23 0x34 0x45} 4

Write 4 values from the list of values to address 0x0

mwr -force 0xe0000000 0x1234

Over-write access protection and write 0x1234 to address 0xe0000000
