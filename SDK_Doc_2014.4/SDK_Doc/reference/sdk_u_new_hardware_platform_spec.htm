Creating a New Hardware Platform Specification

To create a new hardware platform specification for application development in SDK, do the following:

  1. Click File > New > Project to open the New Project wizard.
  2. Select Xilinx > Hardware Platform Specification.
  3. Click Next.
  4. In the Project name field, type a name for the Hardware Platform Specification project.
  5. Select the Hardware Platform Specification XML file.

Note: Hardware Platform Specification XML can be exported by either Project Navigator or Xilinx Platform Studio (XPS).

  1. Click Finish to create your hardware platform specification project.
