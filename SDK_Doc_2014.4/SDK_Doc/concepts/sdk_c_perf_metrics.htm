System Performance Metrics and Graphs

Performance analysis is divided into five tabs:

  • The APU Performance Summary tab shows the metrics related to the APU. See APU Performance Summary.
  • The PS Performance tab shows the graphs reflecting how the metric is changing with time. See PS Performance.
  • The PL Performance tab shows the graphs reflecting how the metric is changing with time. See PL Performance.
  • The MicroBlaze Summary and MicroBlaze Performance tabs show the metrics and corresponding graphs. These tabs are only visible in designs that use a MicroBlaze processor.
