Xilinx Embedded Hardware

Xilinx® FPGAs provide customizable silicon on top of which different kinds of hardware can be created. The Xilinx Vivado® Design Suite provides tools to create custom embedded hardware on Xilinx FPGAs.

In a typical workflow, a hardware designer creates the embedded hardware. The hardware developer generates the Hardware Platform Specification file, which contains all relevant hardware information for the software developer. The software developer uses this information to create the software that runs on the embedded hardware platform.
