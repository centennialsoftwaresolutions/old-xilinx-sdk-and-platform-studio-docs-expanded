Working with software projects

When working with your software project, you will use functionality provided by the Eclipse Workbench and CDT C/C++ Development Tool environment. Online help topics from both user guides are available in SDK. To access them, select Help > Help Contents.

Eclipse Workbench User Guide topics you may find helpful include:

  • Tasks > Working with views and editors
  • Tasks > Working with projects, folders and files
  • Tasks > Working with local history

C/C++ Development User Guide topics you may find helpful include:

  • Tasks > Working with C or C++ project files
  • Tasks > Write code
  • Tasks > Running and debugging projects
  • Reference > C/C++ Views and Editors
  • Reference > C/C++ Menubar
  • Reference > C/C++ Toolbar
  • Reference > C/C++ Preferences
  • Reference > C/C++ Properties
