Debugging with System Debugger (TCF)

The System Debugger (TCF), from open source, is customized and integrated with Xilinx® SDK.

Apart from the Xilinx-customized TCF debugger, the XMD/GDB debug flow that is supported in earlier versions of the tool can still be used. Updates in XMD and enhancements to cable infrastructure better support the integration of multiple JTAG commands and the usage of the System Debugger (TCF) agent for JTAG communication.
