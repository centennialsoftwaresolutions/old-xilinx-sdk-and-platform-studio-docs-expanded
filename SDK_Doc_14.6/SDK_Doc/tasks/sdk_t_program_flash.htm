Programming parallel Flash memory

To program the parallel Flash memory:
  1. Select Xilinx Tools > Program Flash Memory.
  2. In the Select Processor dialog box that opens, select the processor instance to which the parallel Flash memory is connected.
  3. In the Image field, specify the executable file, bitstream, or data file to write to Flash.
  4. If you are programming an executable file (in ELF format) and would like to use a bootloader to boot the processor after reset, click to select the Convert ELF to bootable SREC format and program check box.
  5. Select the Flash memory using the Controller drop-down list in the Flash Memory Settings area.
  6. If you want the file to be written at an offset in the Flash memory, specify the offset using the Program at offset field.
    Note: If the target flash device is a Xilinx Platform Flash, the flash programmer cannot query the device when it comes out of reset. In this case click the Target Flash Device is a Xilinx Platform Flash check box.
  7. Specify any memory that will be used by the Flash programmer as a scratch memory using the Controller drop-down list in the Working Memory Settings area.
    Note: The current contents of this memory will be overwritten.
  8. Click OK. SDK programs the Flash with the file.
