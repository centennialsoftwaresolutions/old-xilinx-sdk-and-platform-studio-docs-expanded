Standalone Application Debug Using Xilinx System Debugger (TCF)

This topic describes how to use the Xilinx®-customized System Debugger (TCF) to debug bare-metal applications.

 


  1. Create a sample Hello World project.


  2. Select the application and click Run > Debug As > Launch on Hardware (TCF).
