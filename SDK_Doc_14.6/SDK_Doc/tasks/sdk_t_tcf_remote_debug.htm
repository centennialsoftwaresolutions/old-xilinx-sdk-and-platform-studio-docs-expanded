Remote Debug with System Debugger (TCF)


Setting Up the Remote System Environment

 

  1. Running hw_server with non-default port (for example: 3122) enables remote connections. Use the following command to launch hw_server on port 3122:
    2. hw_server -s “TCP::3122;Name=Project hello_world Hardware Server”
  2. Make sure your board is correctly connected.

  3. In a cmd window, check the IP Address:



Setting Up the Local System for Remote Debug

 


  1. Launch SDK.


  2. To configure the JTAG settings, click Xilinx Tools > Configure JTAG Settings.



  3. Set the JTAG cable type as Xilinx Hardware Server and provide Hostname and Port details..



  4. Select the application to debug remotely.



  5. Click Run > Debug As > Launch on Hardware (System Debugger).
