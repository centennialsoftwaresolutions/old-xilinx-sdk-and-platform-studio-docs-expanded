Debugging with System Debugger (TCF)

The Xilinx-customized System Debugger is dervied from open-source tools and integrated with Xilinx SDK.

Apart from the Xilinx-customized System Debugger, the XMD/GDB debug flow that is supported in earlier versions of the tool can still be used.

You will notice improved performance in this release because of the updates in XMD and underlying cable infrastructure that supports integration of multiple JTAG commands and the use of the System Debugger agent for JTAG communication.

System Debugger support is integrated only for the ARM over Digilent cable. You can apply the SDK-XMD flow for the Xilinx USB cable, which is still supported as released in the earlier versions.
