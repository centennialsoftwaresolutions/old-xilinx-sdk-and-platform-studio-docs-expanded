Standalone Application Debug Using Xilinx TCF Debugger

This topic describes how to use the Xilinx Customized TCF debugger to debug bare-metal applications.

  1. Follow the steps mentioned in Environment Setup..
  2. Create a sample Hello World project.


  3. Select the application and right-click on it.


  4. Click on “Launch on Hardware (TCF)” to launch the Xilinx Customized TCF debugger.
