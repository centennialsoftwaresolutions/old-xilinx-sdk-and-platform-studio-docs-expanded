Selecting a run/debug/profile configuration

You can select an existing run configuration to use to run your program.

To select a run configuration:

  1. In the C/C++ Projects view, select a project.
  2. Select Run > Run Configurations or Run > Debug Configurations.
  3. In the Debug Configurations dialog box, expand Xilinx C/C++ ELF and select a configuration.
  4. Select a run or debug configuration from the Configurations list.
  5. Click Run or Debug.
