Debugging with TCF

The TCF debugger from open source is customized and is integrated with Xilinx SDK.

Apart from the Xilinx Customized TCF debugger, the XMD/GDB debug flow that is supported in earlier versions of the tool can still be used. Users will see an improved performance in this release due to the updates in XMD and underneath cable infrastructure to support coalescing of multiple JTAG commands and the usage of TCF agent for JTAG communication.

The TCF debugger support is integrated only for ARM over Digilent cable. Please look at the recommended debugger usage page to know when to use which debugger.
