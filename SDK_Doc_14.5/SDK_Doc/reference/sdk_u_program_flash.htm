Program Flash Memory dialog box

You can use the Program Flash Memory dialog box to program the parallel Flash memory in the system.

The Program Flash Memory dialog box contains the following items:
Name Function
Image In this field, specify the name of the file to write to the Flash memory.
Convert ELF to bootloadable SREC format and program Select this check box to have SDK automatically convert the file to the selected bootloadable format when programming Flash.
Flash Memory Settings Select the Flash Memory and Offset to Program.

Controller

 Select the Flash memory to program.

Address & Size

 Properties of the Flash Memory selected to program.

Program at offset

 Offset from the Flash Base address, where the file should be programmed.

Target Flash Device is a Xilinx Platform Flash

 Select this check box if the flash on the board is a Xilinx Platform Flash.
Working Memory Settings Specify the memory to use as scratch memory by the in-system programmer.
