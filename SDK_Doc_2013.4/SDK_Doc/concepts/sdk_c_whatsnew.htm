What’s New in SDK

The following are new features in SDK 2013.4:

  • Added a new “Zynq DRAM tests” application project template.
  • Added debug support for new MicroBlaze version 9.0
  • System debugger access to PL address register does not require to set HW_SERVER_ALLOW_PL_ACCESS environment variable.
  • Introduced a new command line Debug utility(bootgen_utility) to debug the bootimage.

For changes in earlier releases, refer to What’s New in SDK 14.7/2013.3 and Earlier Releases.
