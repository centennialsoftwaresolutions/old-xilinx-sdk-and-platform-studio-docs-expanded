Exporting Resources to an Archive File

You can use the Export wizard to export resources from the Workspace to an archive file in the file system.

  1. In one of the navigation views, select the resources that you want to export.
  2. Select File > Export.
     The Export wizard opens.
  3. Select General > Archive file.
  4. Click Next.
     By default, SDK exports the selected resources along with their children. You can modify the set of resources to export by selecting them in the Archive file pane. Select items individually, or use the Select All and Deselect All buttons. You can filter the file types using the Select Types button.
  5. In the To archive file field, specify the path and name of the archive file into which you want to export the selected resources.
  6. Click Finish.
