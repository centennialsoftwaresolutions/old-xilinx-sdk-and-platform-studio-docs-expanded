Building Projects

When building projects, you will use functionality provided by the Eclipse Workbench and CDT C/C++ Development Tool environment. Online help topics from both user guides are available in SDK. To access them, select Help > Help Contents.

Eclipse Workbench User Guide topics you might find helpful include:

  • Tasks > Working with projects, folders and files (all topics)

C/C++ Development User Guide topics you might find helpful include:

  • Tasks > Building Projects > Renaming a project
  • Tasks > Building Projects > Defining Project Build settings
  • Tasks > Building Projects > Filtering Errors
  • Tasks > Building Projects > Adding Include paths and symbols
  • Tasks > Building Projects > Building Manually
  • Tasks > Building Projects > Removing Build Automatically
  • Tasks > Building Projects > Autosaving on a build
  • Tasks > Building Projects > Creating a make target
  • Tasks > Building Projects > Customizing the Console view
  • Tasks > Building Projects > Viewing and managing compile errors
