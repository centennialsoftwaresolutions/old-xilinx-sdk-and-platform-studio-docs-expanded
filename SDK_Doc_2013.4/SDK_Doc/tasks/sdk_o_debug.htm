Debugging a Program

You use a debug configuration to debug a program on a target. If you have an existing debug configuration, you can reuse it. Otherwise, you must create a new debug configuration to debug your program.

  1. Click Run > Debug Configurations.
  2. In the Configurations list, select a debug configuration.
  3. Click Debug.

SDK switches to the debug perspective and starts downloading and debugging the program. To view the program output, use the STDIO Connection tab to use SDK console.

The debug perspective offers a rich set of debug views to monitor the variables, memory and register contents, breakpoints, source code, etc of the program. For more information on Debug Perspective refer CDT User Guide.
