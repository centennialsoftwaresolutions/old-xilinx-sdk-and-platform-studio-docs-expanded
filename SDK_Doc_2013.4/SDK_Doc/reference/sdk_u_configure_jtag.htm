Configure JTAG Settings Dialog Box

In the Configure JTAG Settings dialog box, specify the JTAG cable to use for communication and the JTAG Device Chain information for the board.

To open the Configure JTAG Settings dialog box, select Xilinx Tools > Configure JTAG Settings.

The Configure JTAG Settings dialog box contains the following functions.

Name

Function

JTAG Cable

Specify the JTAG Cable settings.

Type

Specify the Cable type. The default setting is “auto”.

Port

Specify the port on the host machine to which the cable is connected.

Frequency

Specify the frequency at which to run the cable.

Other Options

Specify the cable name and arguments for third party cable types in this field. Specify the full cable option:

-cable type xilinx_plugin modulename <cable name> {modulearg <cable args>}

JTAG Device Chain

Specify the JTAG Device chain information for the board.

Automatically Discover Devices on JTAG Chain

Use this option for SDK to automatically detect the devices in the chain. This is recommended for most users.

Manual Configuration

Use this option if the auto-discover failed.

FPGA?

Specify the FPGA device for communication by SDK.

Device Name

Name of the device

ID Code

The JTAG ID of the device

IR Length

The JTAG IR Length of the device

Add/Delete/Move Up/Down

Use these options to specify the devices in the chain.
