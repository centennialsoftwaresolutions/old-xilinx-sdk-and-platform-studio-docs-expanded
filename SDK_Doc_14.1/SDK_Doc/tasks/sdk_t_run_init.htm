Specifying initialization tasks

To configure the XMD target options, do the following:

  1. In the C/C++ Projects view, select a project.
  2. Select Run > Run Configurations or Run > Debug Configurations.
  3. In the Configurations box, expand Xilinx C/C++ ELF.
  4. Select a run or debug configuration.
  5. Click the Device Initialization tab.



  6. In the Reset Type drop-down list, select the reset type.
  7. If you do not want to download the ELF program file, select Do not download program to memory.
  8. If you want to download any data files, add the data files to the download list, along with the and start address at which you want to download them.
  9. To enable program debugging in safe mode, click to select Enable safemode. For safe mode usage information, see also the Embedded System Tools Reference Manual UG111 topic covering the XMD safemode command.
  10. Click Apply to save the settings.
