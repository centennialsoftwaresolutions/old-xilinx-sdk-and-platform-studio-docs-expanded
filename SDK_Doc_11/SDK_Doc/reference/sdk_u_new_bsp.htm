New Board Support Package Project dialog box

In the New Board Support Package Project page of the New Project wizard, you can create a new board support package project. To open this page, select File > New > Project, select Board Support Package, and click Next.

Select a name for the project, the processor, and the platform type. You can use the default location, or specify a path for your project by deselecting the Use Default Location checkbox and entering the new path in the Location text box.

The New Board Support Package Project page contains the following items:
Name Function
Name Specify a name for the project.
Processor Select the processor for which you want to build the BSP.
Board Support Package Type Select the BSP type from the available BSPs for the processor.
Use Default When this check box is selected, SDK creates the new project in the default workspace location.
Directory If Use Default Location is not selected, specify the location where the project is to be created.
