Flash Memory Programmer dialog box

You can use the Flash Memory Programmer dialog box to program the parallel flash memory in the system.

The Flash Memory Programmer dialog box contains the following items:
Name Function
File to Program In this field, specify the name of the file to write to the Flash memory.
Automatically convert file... Select this check box to have SDK automatically convert the file to the selected bootloadable format when programming Flash.
Flash Memory Select the Flash Memory and Offset to Program.

Instance Name

 Select the Flash memory to program.

Address, Size & Bus Width

 Properties of the Flash Memory selected to program.

Program at Offset

 Offset from the Flash Base address, where the file should be programmed.
Scratch Memory Specify the memory to use as scratch memory.
Create Flash Bootloader Application Select this check box to view options for specifying parameters for the bootloader application.

Software Platform

 Select from the list of software platform for the processor in the workspace.

SW Application Project

 Name of the project in the workspace.

Bootloader File Format

 Displays the format of the bootloader file.
