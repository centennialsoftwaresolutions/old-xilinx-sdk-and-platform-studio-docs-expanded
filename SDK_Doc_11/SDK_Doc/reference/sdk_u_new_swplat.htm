New Software Platform Project dialog box

In the New Software Platform Project page of the New Project wizard, select a name for the project, the processor, and the platform type. You can also enter a new path for your project by unselecting the Use default checkbox and specifying the new path in the Directory field.

The New Software Platform Project page contains the following items:
Name Function
Name In this field, specify the name of the project.
Processor Select the processor for which you want to build the software platform.
Platform Type Select the platform type from the list of available types.
Use default When this check box is selected, SDK creates the new project in the default workspace location.
Directory If Use default check box is not selected, use this field to specify the location where the project is to be created.
