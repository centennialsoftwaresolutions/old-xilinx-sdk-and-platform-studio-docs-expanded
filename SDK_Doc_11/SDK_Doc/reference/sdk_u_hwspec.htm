Hardware Specification File dialog box

When you create a new SDK workspace, use the Hardware Specification File dialog box to specify the hardware specification file to use for creating software projects.

In the Hardware Specification File field, type or browse to the hardware specification file of the hardware design generated in Xilinx® Platform Studio (XPS).

Re-targeting software projects

In an existing design, you can re-target the software projects in a workspace to build and run on a different hardware design. Select Hardware Design > Import Hardware Design to open the Hardware Specification File dialog box and specify the new hardware specification file.

The Hardware Specification File dialog box contains the following functions:
Name Function
Current Hardware Design The current hardware specification file used.
Hardware Specification File The new hardware specification file to use to re-target your projects.
Show Hardware Design Differences Click this button to view the differences in the hardware designs.
Preview Software Changes Expand this section to view the changes to the software projects.
