Project Type:
Displays the type of project that is built by the selected build configuration.
Configuration:
Specifies the selected build configuration from the list provided. Refer the Selecting a project type  section for more details.
Manage...
Adds, removes, renames and converts configurations.
Configuration Settings, Tool Settings
Sets build options for compilation and linking.
  • Command field displays the compiler or linker used for building the project.
  • All Options field displays the compiler or linker options that will be used for building the project. The options are based on what you have specified in other fields.
  • MicroBlaze-Specific Options or PowerPC-Specific Options is a read-only field that displays the processor configuration specific compiler options. These options are automatically inferred from the hardware specification file.
  • Software Platform Options is a read-only field that displays the compiler or linker options that were set in software platform configuration dialog.
