Software Platform Settings dialog box

In the Software Platform Settings dialog box, you can select the drivers and libraries to use in the software platform and configure the parameters. To open this dialog box, select Tools > Software Platform Settings.

Software Platform page

Use the Software Platform page of the Software Platform Settings dialog box to customize the processor settings and the OS and Libraries options.

Processor Settings

The Processor Settings area of the Software Platform page contains the following functions:
Name Function
CPU Driver Specifies the name of the CPU driver.
CPU Driver Version Specifies the version of the CPU driver.
Processor Parameters Displays a list of parameters for the CPU driver.
Name
 Name of the Parameter.
Current Value
 User specified value. If you do not specify a value, this field contains the default value.
Default Value
 Default value of the parameter.
Type
 Parameter value type, such as string, integer, bool, array, or peripheral instance.
Description
 Description of the parameter.

OS Version and Libraries Selection

The OS Version and Libraries Selection area of the Software Platform page contains the following functions:
Name Function
OS Specifies the type of software platform.
Version Select the version of software platform.
Details Table Displays the details for the Libraries.
Use
 Click to select Library checkboxes to include libraries in the software platform.
Library
 The name of the library.
Version
 The version of the library.
Description
 The description of the library.

Click the Download Partner Library Definition Files button to open a Xilinx page containing partner information for various versions of libraries in your default browser.

OS and Libraries page

Use the OS and Libraries page of the Software Platform Settings dialog box to configure various parameters of the operating system and its constituent libraries.

The OS and Libraries page contains the following functions:
Name Function
Name Name of the Parameter
Current Value User specified value. Contains the default value if not changed by the user
Default Value Default value of the parameter
Type Parameter value type. The values can be string, integer, bool, array, peripheral_instance, etc
Description Description of the parameter

Drivers page

Use the Drivers page of the Software Platform Settings dialog box to select the drivers to use for the peripherals in your system.

The Drivers page contains the following functions:
Name Function
Peripheral Name of the hardware peripheral.
HW Version Version of the hardware peripheral.
Instance Name of the hardware peripheral instance.
Driver Name of the software driver used. If you don't want to include this peripheral in the software platform, select "none" as the driver name.
Version Version of the software driver used.
