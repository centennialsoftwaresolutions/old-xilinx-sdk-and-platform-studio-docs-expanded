JTAG settings

SDK communicates to the hardware over the JTAG interface for debugging, running, and profiling an application and for programming FPGA and Flash. You are also required to use the Xilinx® JTAG cable for communication. The following settings are required for JTAG communication.

  • JTAG Cable - Xilinx provides different JTAG cable types such as Xilinx Parallel Cable and Xilinx USB Cable. You can connect multiple cables to your host machine. Use this option to select the JTAG cable to use for communication and customize the configurations. The default type of auto is recommended when only one cable is connected to your host machine.
  • Board JTAG Device Chain Definition - The hardware board can have a series of JTAG devices chained together. SDK requires the JTAG Instruction Register (IR) length, IDCODE, and position in the chain of each JTAG device for communication. SDK maintains a list of these values for Xilinx devices, but you must provide the JTAG information for non-Xilinx devices present on the board.
