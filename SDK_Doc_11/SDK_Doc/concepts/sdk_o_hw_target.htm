Xilinx embedded hardware

Xilinx® FPGAs provide customizable silicon on top of which different kinds of hardware can be created. The Xilinx Embedded Development Kit (EDK) provides tools to create custom embedded hardware on Xilinx FPGAs. EDK consists of the following tools:

  • Xilinx Platform Studio (XPS): A graphical embedded hardware creation tool.
  • Embedded IP: A set of IP cores such as Ethernet MACs, UART controllers, and memory controllers that make it easy to interface to external peripherals.
  • Xilinx Software Development Kit (SDK): The Eclipse-based software development toolkit that you are using.
In a typical workflow, a hardware designer creates embedded hardware using XPS. The hardware developer exports the relevant hardware information to the software developer. The software developer uses this information to create software that would run on the embedded hardware platform.
