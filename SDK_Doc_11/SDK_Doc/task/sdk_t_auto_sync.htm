Auto-synchronization of software projects

When the Hardware Specification file used by the SDK workspace changes, SDK automatically detects this change and synchronizes the software projects to the changes in the hardware design. The Hardware Design Changes Detected dialog box opens.

In this dialog box, you can do the following:

  • Click Show Hardware Changes to view the changes to the hardware design. This opens an HTML page.
  • In the Software Changes list, review the changes that will be made to software projects.
  • Click OK. SDK commits the changes to software projects and starts rebuilding them.
