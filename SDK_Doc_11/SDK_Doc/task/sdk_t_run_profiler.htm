Specifying Profiler configuration

To configure options for the Profiler, do the following:
  1. In the C/C++ Projects view, select a project.
  2. Select Run > Run.
  3. In the Configurations box, expand Xilinx C/C++ ELF.
  4. Select a run configuration.
  5. Click the Profiler tab.



  6. Do the following:
    • Click to select the Enable Profiling check box.
    • Click to select the Show Profile Results Immediately check box.
    • In the Profiling Parameters for Hardware Targets area, specify the histogram sampling frequency and bin size, and the scratch memory to use for profiling.
  7. Click Run to profile the application
