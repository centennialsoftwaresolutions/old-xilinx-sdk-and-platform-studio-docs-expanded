Specifying the compiler options

You can specify the compiler options for Managed Make projects.

Specifying debug and optimization compiler flags

Based on the build configuration selected, SDK assigns a default optimization level and debug flags for compilation. You can change the default value for your project.

To set properties for your project:

  1. Right-click your managed make project and select Properties. Alternatively, to set properties for a specific source file in your project, right-click a source file within your standard make project and select Properties.
  2. Click C/C++ Build.
  3. In Tool Settings tab, select Debug and Optimization to change the debug and optimization levels.

Specifying Miscellaneous Compiler Flags

You can specify any other compiler flags not covered in the Tool Settings for program compilation.

To set properties for your project:

  1. Right-click your managed make project and select Properties. Alternatively, to set properties for a specific source file in your project, right-click a source file within your standard make project and select Properties.
  2. Click C/C++ Build.
  3. In the Tool Settings tab, select the Miscellaneous > Other Compiler Options to Append to specify compiler flags.
