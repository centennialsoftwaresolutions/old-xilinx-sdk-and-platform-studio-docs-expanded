Creating or editing a Run/Debug/Profile configuration

You can run an application by right-clicking the application name in the C/C++ Projects View and clicking Run As > Run on Hardware. This creates a default run configuration and runs the application on the target. Similarly, you can debug an application by right-clicking the application name in the C/C++ Projects View and clicking Debug As > Debug on Hardware. You can also configure a run or debug environment with which to run or debug your application.

The topics listed below describe how to create a run, debug, or profile configuration.

The Run and Debug dialog boxes contain the following tabs when a configuration is selected:

  • Main
  • XMD Target Connection
  • Initialization
  • Remote Debug
  • Profiler
  • Initial Breakpoint
