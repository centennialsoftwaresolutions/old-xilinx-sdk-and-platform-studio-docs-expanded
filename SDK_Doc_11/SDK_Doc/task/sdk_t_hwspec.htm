Importing a hardware specification file

To import a hardware specification file

SDK requires you to specify a hardware specification file before you can start developing C/C++ application projects. When SDK starts for the first time with a given workspace, it opens the New Hardware Specification File dialog box prompting you to specify the location of the hardware specification file. You can also specify a new hardware specification file at any time during application development by selecting Hardware Design > Import Hardware Design, which opens the New Hardware Specification File dialog box.


To view a hardware design report

You can view the design report of the hardware system by selecting Hardware Design > View Design Report. SDK opens the HTML design report in a browser, displaying the following information about the hardware design:
  • The block diagram of the design
  • The list of processors in the design, the processor memory map, and the  list of peripherals in the processor sub-system
  • The configurable parameters of each peripheral in the design
  • The IP Datasheet of each peripheral in the design
