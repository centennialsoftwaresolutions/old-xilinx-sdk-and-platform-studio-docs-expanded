Creating a board support package

To create a new Board Support Package (BSP):
  1. Click File > New > Board Support Package.
  2. In the Project name field, type a name for the instance of the BSP, such as linux_0.
  3. From the Processor drop-down list, select the processor for which you want to generate the BSP.
  4. From the Board Support Package Type drop-down list, select the BSP type from the available types.
  5. Select the location for the BSP project files. To use the default location, as displayed in the Directory field, leave the Use default check box selected. Otherwise, click to unselect the check box, then type or browse to the directory location.
  6. Click Finish. The wizard creates a new BSP and displays it in the SDK Navigator pane.
