Configuring a board support package

To configure an existing Board Support Package (BSP) project do the following:

  1. In the SDK Navigator pane, right click on the BSP you wish to configure and click on Board Support Package Settings.
  2. This will bring up the Board Support Package Settings Dialog.
  3. In this dialog box, you will be able to configure various parameters of the OS.


You can cancel out of the dialog at any time. If you are done with all the settings you want to make, click OK to commit your changes.

If the Build Automatically option is selected in the Project menu, your BSP is regenerated with your new settings applied.
