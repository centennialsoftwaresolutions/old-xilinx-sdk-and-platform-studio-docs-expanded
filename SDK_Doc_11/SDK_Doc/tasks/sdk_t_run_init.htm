Specifying initialization tasks

To configure the XMD target options, do the following:
  1. In the C/C++ Projects view, select a project.
  2. Select Run > Run or Run > Debug.
  3. In the Configurations box,expand Xilinx C/C++ ELF.
  4. Select a run or debug configuration.
  5. Click the Initialization tab.



  6. Do the following:
    • In the Reset Type drop-down list, select the reset type.
    • If you want to download any data files, click to select Download Program. Then add the data files to the download list, along with the and start address at which you want to download them.
    • To enable program debugging in safe mode, click to select Enable safemode.
  7. Click Apply to save the settings.
