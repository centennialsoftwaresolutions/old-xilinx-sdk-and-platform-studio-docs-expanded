Exporting resources to the file system

You can use the Export wizard to export resources from the Workspace to the file system.

  1. In one of the navigation views, select the resources that you want to export.
  2. Select File > Export. The Export wizard opens.
  3. Select File system and click Next.
  4. By default, SDK exports the selected resources along with their children. You can modify the set of resources to export by selecting them in the Archive file pane. Select items individually, or use the Select All and Deselect All buttons. You can filter the file types using the Select Types button.
  5. In the To directory field, specify the path and name of the directory into which you want to export the selected resources.
  6. Click Finish.

Note: You can also export folders and files by dragging them from one of the navigation views to the file system and dropping them in the file system, or by copying and pasting them.
