Profiling with TCF Debugger

  1. Select the application you want to profile.
  2. Select Run > Debug As > Launch on Hardware (System Debugger).
  3. When the application stops at main, open the TCF profiler view by selecting Window > Show View > Other > Debug > TCF Profiler.



  4. Click the button to start profiling. Alternately, you can select the Aggregate Per Function option in the Profiler Configuration dialog box.



  5. Resume your application. You can now view the profile data.
