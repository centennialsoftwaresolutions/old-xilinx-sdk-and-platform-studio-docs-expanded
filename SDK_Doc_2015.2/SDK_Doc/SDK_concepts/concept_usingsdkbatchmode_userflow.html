User Flow

The following user flow describes how you can use the SDK batch mode.

  1. Start sdk batch mode using the command:

    xsct

  2. Set the workspace (where you want to create SDK projects) using the command:

    sdk set_workspace <Workspace Path>

  3. If you have any local repositories, set the user repository path using the command:

    sdk set_user_repo_path <Repository Path>

  4. Create a hardware project using the command:

    sdk create_hw_project -name <hw project> -hwspec <.hdf/.xml path>

  5. Create a BSP for the hardware project using the command:

    sdk create_bsp_project -name <bsp project name> -hwproject <hw project> -proc <processor> -os <OS Name>

  6. Create an application project using the command:

    sdk create_app_project -name <Application Name> -hwproject <hw Project> -proc <processor> -os <Os Name> -lang <Language> -app <Application Template> -bsp <Bsp Project>

  7. Build the projects using the command:

    sdk build_project

  8. Clean the projects using the command:

    sdk clean_project

You can also open the projects created in SDK GUI mode with the command:

xsdk -workspace <workspace path>
