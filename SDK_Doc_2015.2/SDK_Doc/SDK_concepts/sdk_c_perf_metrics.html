System Performance Metrics and Grabs

Performance analysis is divided into five tabs:

  • The APU Performance Summary tab shows the metrics related to the APU. See ../SDK_references/sdk_u_apu_performance_summary.html#Rcz77144.
  • The PS Performance tab shows the grabs reflecting how the metric is changing with time. See PS Performance.
  • The PL Performance tab shows the grabs reflecting how the metric is changing with time. See PL Performance.
  • The MicroBlaze Summary and MicroBlaze Performance tabs show the metrics and corresponding grabs. These tabs are only visible in designs that use a MicroBlaze processor.
