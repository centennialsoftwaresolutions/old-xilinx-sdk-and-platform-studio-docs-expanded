Xilinx Software Command-Line Tool (XSCT)

The Xilinx software command-line tool is based on Tcl and is delivered in the SDK install. This tool provides the Tcl prompt and you can use all of the supported commands.

The Xilinx software command-line tool is a scriptable commandline interface to run SDK commands, XSDB commands and HSI commands. XSCT can be started from the Start menu or by executing the xsct.bat file, available in the <SDK installation directory>/bin folder. All the commands supported by XSCT are grouped under their respective categories.
