Project Types

You can create a standard make C or C++ project or a managed make C or C++ project.

  • SDK managed C or C++ project (recommended for beginners)

    An SDK managed make project automatically generates the makefiles for you. These files are required for your managed make projects to build successfully.

  • Standard make C or C++ project

    You must create a makefile or use an existing makefile in order to build your project .
