sdk create_hw_project

Description

You can use this command to create the hardware platform project.

Note: Make sure that you set the workspace before using this command.

Usage

 sdk create_hw_project [options]

Create a hardware project.

Options

 -name <project-name>

Project name that should be created.
-hwspec <HW specification file>

HW specification file for creating a hardware project.

This option is valid only for creating hardware projects.
