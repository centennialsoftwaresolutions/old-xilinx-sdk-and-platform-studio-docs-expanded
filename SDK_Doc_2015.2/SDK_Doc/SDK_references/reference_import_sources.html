sdk import_sources

Description

Import sources to an application project.

Usage

 sdk import_sources [options]

Import sources from a specified path to an application project in the workspace.

Options

 -name <project-name> Application project to which the sources should be imported.
-path <source-path>

Path from which the source files should be imported.

All the files/directories from the specified source path are imported to an application project.

Note: Any existing source files in the application project are deleted before importing the new sources.
