Program input and output

If the software program input/output is set to use a serial interface, you can redirect input and output to your console, use the terminal plugin program included with SDK, or use any program such as hyperterminal or teraterm to view the program output.

To connect input and output to your console, click to select the Connect process input & output to a terminal check box in your run or debug configuration. For more information, refer to Creating a run/debug/profile configuration.

To use the terminal plugin program included with SDK, select Window > Show View > Terminal and click OK. A terminal view appears in your workspace and you can configure its settings.

If the software program output is set to use JTAG Uart or MDM peripheral, type the command terminal on the XMD console to have SDK launch a window in which you can view the program output.
