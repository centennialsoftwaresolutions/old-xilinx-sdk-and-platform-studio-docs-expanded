Creating or editing a run/debug/profile configuration

You can run or debug an application by right-clicking the application name in the C/C++ Projects View and clicking Run As > Launch on Hardware. This creates a default run configuration and runs the application on the target. You can also configure a run environment with which to run your application.

The Run/Debug dialog box contains the following tabs when a configuration is selected:

  • Main
  • XMD Target Connection
  • Initialization
  • Remote Debug
  • Profiler
  • Initial Breakpoint
