Creating a run/debug/profile configuration

You can create a customized run, debug, or profile configuration and save it for reuse.

To create a run configuration:

  1. In the C/C++ Projects perspective, select a project.
  2. Click Run > Run Configurations or Run > Debug Configurations.
  3. In the Configurations list, click Xilinx C/C++ ELF.
  4. Click New. The name of the new project is displayed in the Configurations list.

  1. To change the default name of the new run/debug configuration, refer to Selecting an application to run/debug/profile.
