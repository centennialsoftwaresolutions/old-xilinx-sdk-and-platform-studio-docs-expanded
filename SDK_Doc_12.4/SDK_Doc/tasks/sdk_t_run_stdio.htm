Specifying STDIO connections

To use the SDK console as STDIO for your application, do the following:

  1. In the C/C++ Projects view, select a project.
  2. Select Run > Run Configurations or Run > Debug Configurations.
  3. In the Configurations box, expand Xilinx C/C++ ELF.
  4. Select a run or debug configuration.
  5. Click the STDIO Connection tab.



  6. Select Connect STDIO to console
  7. If the BSP of your application uses Serial UART as the STDIO peripheral, specify the serial port to use on your host machine and BAUD rate.
  8. If the BSP of your application uses MDM as the STDIO peripheral, select JTAG UART as the port type.
  9. Click Apply to save the settings.
