rrd

Register Read

Syntax

rrd [options] [reg]

Read registers for active target

Options

Option                       Description
-defs Read register definitions instead of values
-no-bits Don't show bit fields along with register values. By default, bit fields are shown, when available

Returns

Register names and values, or register definitions if successful. Error string, if the registers can't be read or if an invalid register is specified.

Example(s)

rrd

Read top level registers or groups 

rrd r0

Read register r0 

rrd usr r8

Read register r8 in group usr
