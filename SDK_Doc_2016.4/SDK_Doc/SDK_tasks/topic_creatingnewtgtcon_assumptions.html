Assumptions

  • Ensure that the target (board) is connected to the remote host.
  • Launch hw_server manually on the remote host:
    1. Take a shell on the remote host.
    2. Source the setup scripts.

      C:/Xilinx/SDK/<version>/settings64.bat (or) /opt/Xilinx/SDK/2015.1<version>/settings64.csh

    3. Run 
      hw_server
      .
