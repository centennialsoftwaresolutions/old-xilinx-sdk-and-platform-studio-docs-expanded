Selecting a Target Development Board in the Base System Builder
In the Board and System Selection page of the Base System Builder, Choose whether you are targeting one of the supported embedded processor development boards available from Xilinx or from one of our partners, or designing a custom board.
Note If you invoke XPS and the Base System Builder (BSB) by means of creating a new embedded processor source in the Xilinx® ISE® Project Navigator, the selection of target boards offered in this BSB page is limited to those containing the same FPGA device that you selected in the ISE software. If the selected FPGA is not featured on any of the supported boards, then you can only target a custom board in this BSB page.
To Select a Target Development Board
When you select a target development board, the BSB is able to determine what devices are on the target board, such as the specific FPGA device, external memories, I/O devices, clock resources, and reset polarity. The following wizard pages are customized based on the information that can be retrieved for this board, minimizing the amount of input that is required.
If you do not find the board you are looking for in the drop-down lists, click Third Party Board Definition Files Download Website in the Links section of the window to see if it is available through a web download.
  1. Select the I would like to create a system for the following development board option.
  2. Select the vendor of the board from the Board Vendor drop-down list.
  3. Select the specific board from the Board Name drop-down list.
  4. Select the revision of the board from the Board Revision drop-down list.
    Note When you select a board, the Board Configuration section of the window is pre-populated for you with the board information.
  5. If the board uses stepping, select the Use Stepping check box and select a value from the drop-down list.
    Note For more information about stepping, refer to Using Stepping in the Base System Builder.
To Select a Custom Board
If the board you are using is custom or not supported by the BSB, you can generate a system for your board. Using the Custom Board option typically requires more input because the wizard can not determine much of the information about your custom board. In subsequent pages, you must provide information such as the FPGA device you are using, external clock frequency and reset polarity, and what external memories and I/O devices you are using. In addition, the User Constraints File (UCF) generated by the BSB does not contain any FPGA pin location information. You must manually enter that information into the UCF file before you can implement this design and run it on actual hardware.
  1. Select the I would like to create a system for a custom board option.
  2. In the Board Configuration area, select the architecture, device, package, and speed grade for the board.
  3. If the board uses stepping, select the Use Stepping check box and select a value from the drop-down list.
    Note For more information about stepping, refer to Using Stepping in the Base System Builder.
  4. Set the Reset Polarity for the board.
  5. Set the reference clock frequency (in MHz).The reference clock is the external clock source on the board you are using. If you selected the Custom Board option or if the board you are using has a configurable clock source (such as a socket for an oscillator), the reference clock frequency is editable. You must ensure that the value you enter here is the frequency of the actual clock source on your board.
Note You will also need to provide information about the available devices for your custom board later in the BSB wizard. For more information, refer to Adding and Configuring Peripherals in the Base System Builder.
The next step is Configuring Your System Type.
