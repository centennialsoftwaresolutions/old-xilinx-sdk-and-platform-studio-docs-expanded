Zynq DRAM Diagnostics Test

The Zynq-7000 AP SoC DRAM diagnostics test is a stand-alone program running on a single Zynq-7000 AP SoC-7000 AP SoC Cortex-A9 processor, executing out of OCM. The program uses the UART for interactive operation.

The full Zynq-7000 AP SoC DRAM Diagnostics Test instructions can be accessed in your SDK installation by navigating to <SDK installation directory>\data\embeddedsw\lib\sw_apps\zynq_dram_test\src.
