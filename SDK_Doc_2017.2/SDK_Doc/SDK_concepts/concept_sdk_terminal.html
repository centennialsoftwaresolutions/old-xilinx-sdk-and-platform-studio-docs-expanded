SDK Terminal

The SDK Terminal view supports connections to remote systems via serial connections. The SDK Terminal view replaces the existing terminal in SDK and can be accessed from the default perspective.

The table below lists all the options available on the SDK Terminal view.

Table 1. SDK Terminal Options
Option Description
Connect to serial port The Connect to serial port dialog box appears. Specify a new serial port or update an existing serial port details.
Remove a serial port from the monitor Click to disconnect the connected port.
Clear console Click to clear the console.
Send Click to send input to the terminal.
Clear Click to clear the console. Similar to using the Clear console toolbar icon.
Note: Two serial port connections are not allowed at the same time. If you want to connect to a new serial port user, you will have to disconnect the existing connection.
