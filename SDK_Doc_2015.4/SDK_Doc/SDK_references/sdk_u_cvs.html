Using SDK with Revision Control Software

This section provides details on how to use SDK in conjunction with version control systems (VCS).

For more details on:

  • Git - Refer Git for Eclipse Users for detailed documentation.
  • Perforce - Refer http://www.perforce.com/resources/tutorials/p4eclipse-plug for detailed documentation and http://www.perforce.com/downloads for download details.
  • SVN - Refer http://www.eclipse.org/subversive/documentation/ for detailed documentation and http://www.eclipse.org/subversive/latest-releases.php for download details.
  • Clear Case - Not supported on 64bit Operating system.
