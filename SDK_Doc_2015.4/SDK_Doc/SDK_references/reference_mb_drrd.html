mb_drrd

Description

Read from MicroBlaze debug register.

Usage

 mdm_drrd [OPTIONS] <cmd> <bitlen>

Read a MicroBlaze debug register available on MDM.

  • <cmd> is 8-bit MDM command to access a Debug Register.
  • <bitlen> is the register width.

Returns hex register value.

Options

 -user <bscan number>

Specify user bscan port number. Default is 2.

Example

 mb_drrd 3 28

Read MB Status register.
