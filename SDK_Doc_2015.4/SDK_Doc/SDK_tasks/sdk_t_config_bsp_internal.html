Configuring a Board Support Package (SDK)

There are a number of ways to launch the Board Support Package Settings dialog box:

  • From the Project Explorer, right-click the BSP project to modify and select Board Support Package settings.
  • In the C/C++ Projects view, select the mss tab for the appropriate BSP, and click the Modify this BSP's Settings button. The BSP project can be selected from the Project Explorer. Click Xilinx Tools > Board Support Package Settings, select the BSP to modify and click OK.

The Board Support Package Settings dialog box consists of the following pages:

  • Overview - see Using the Board Support Package Overview Page
  • Board Support Package Configuration - see Using the Board Support Package Settings Page
  • Drivers - see Using the Board Support Package Drivers Page
  • Driver Configuration - see Using the Board Support Package Settings Driver Configuration Page
