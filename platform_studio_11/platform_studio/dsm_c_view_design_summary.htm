Design Summary Overview
The Design Summary allows you to quickly access design overview information, reports, and messages. By default, the Design Summary appears in the Workspace when you open a project, and it displays information specific to your targeted device and software tools. The panes on the left side of the Design Summary allow you to control the information displayed in the right pane, as described in the following sections.
Note To change the default settings, refer to Opening and Navigating the Design Summary and Showing and Hiding Messages and Reports.
Viewing Design Overview and Summary Information
To view high-level information about your design in the right pane of the Design Summary, click the Summary link in the upper left pane, or click the Return to Summary toolbar button . In the lower left pane, select the following options to show additional information about your design:
  •  Project Properties
    •  Enable Enhanced Design Summary shows the number of errors and warnings for the entire project and for each of the Detailed Reports.
      Note You must select this option if you want to use the Show Warnings and Show Errors options.
    •  Display Incremental Messages shows the number of new messages for the most recent software run. You must enable this option, and then run the software to show the number of new messages.
    •  Enable Message Filtering shows the number of messages you filtered. You must enable this option, filter messages, and then run the software to show the number of filtered messages.
  •  Optional Design Summary Contents
    •  Show Clock Report shows clock information from place and route.
    •  Show Failing Constraints shows failed timing constraints.
    •  Show Warnings shows unfiltered warnings the software encountered.
    •  Show Errors shows errors the software encountered.
    •  Show Partition Data shows the Partition Summary, which lists each partition and the status for each, including whether timing was met. You can click a Partition name to open the Partition Report. This link only applies if your design uses Partitions. For more information, refer to Partitions Overview.
Note As the software processes your design, you can reload the data displayed in the right pane. Right-click and select Reload, or click the Reload toolbar button .
Viewing Design Overview Information, Messages, and Reports
In addition to Summary information, the upper left pane of the Design Summary provides access to design overview information, messages, and reports. To view this information in the right pane of the Design Summary, click a title in the upper left pane (for example, Timing Constraints, Synthesis Messages or Synthesis Report). For more information, see Viewing Messages and Reports.
Reports and messages are updated as the software processes your design. Status is indicated by the following icons:
  •  Up-to-date
    The software ran successfully, and the information reflects the latest software run.
  •  Out-of-date
    The software must be rerun to bring the information up-to-date.
  •  Not yet generated
    The software must be run to generate this information.
  •  Location not defined
    The file associated with this information is located outside of the project directory. For more information, see Defining Locations for Messages and Reports.
Note To reload the information displayed in the right pane, right-click and select Reload, or click the Reload toolbar button .
Cross Probing from the Design Summary
When a report is open in the right pane of the Design Summary, right-click a timing constraint link and select Show in Timing Analyzer to cross probe to the Timing Analyzer.
Note You can also cross probe from other links in reports and message lists. Hold your cursor over the link to see whether it is a cross probing link. For more information, see Cross Probing.
